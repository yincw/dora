## 自动样板生成

```
npm run gen dirName fileName
```

运行以上命令会生成以下样板：

- `src` 目录下创建 `dirName` 目录，生成 `fileName` TS 文件（函数主体）
- `src` 目录下 `index.ts` 写入记录：导入，导出
- `__tests__` 目录下创建 `fileName` TS 文件（测试用例）
